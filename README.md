## Tasks Covered

### Task 1: Basic Operations

* Accepts two integer inputs from the user
* Performs:

  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Modulus
  * Exponentiation
* Displays results using formatted output (f-strings)

---

### Task 2: Lists and Arrays

* Created a list with 10 elements
* Performed:

  * Length calculation
  * Finding maximum and minimum
  * Adding and removing elements
  * Sorting (ascending & descending)
* Converted list to NumPy array and calculated:

  * Mean
  * Median
  * Standard Deviation

---

### Task 3: Dictionaries and Sets

* Created a dictionary with student details
* Iterated and printed key-value pairs
* Added a new key (`grade`)
* Demonstrated:

  * Set uniqueness
  * Union
  * Intersection
  * Difference

---

### Task 4: File Handling

* Created a file `student_data.txt`
* Stored student records (name, course, marks)
* Read file and displayed students with marks > 75
